## 🎯 API Overview

### Base URL
```
http://your-domain.com/api
```

### Features
- 🔐 JWT Authentication
- 💰 Pemasukan (Income Management)
- 💸 Pengeluaran (Expense Management)
- 📄 Tagihan (Bill Management)
- 💳 Hutang (Debt Management)
- 👥 Pertemanan (Friend Management)
- 🏷️ Kategori (Categories)
- 📊 Dashboard Statistics
- 💵 Batas Harian (Daily Limit)

### Key Features
- ✅ Filter by Period (bulan_ini, minggu_ini, custom, semua)
- ✅ Pagination Support
- ✅ Auto Pengeluaran from Tagihan
- ✅ Friend Integration for Hutang
- ✅ Multiple Payment Methods
- ✅ Recurring Bills
